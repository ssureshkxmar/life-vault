## 🚀 Overview

This project is a complete career guidance system that works like a **digital career counselor**.  
It analyzes a student’s **aptitude, interests, skills, and personality (DMIT)** to suggest the most suitable career paths.

The platform combines **AI + Data Science + Analytics** into one single solution.

---

## 🎯 Key Features

- 🤖 AI-based career prediction with match percentage
- 🧠 DMIT brain analysis (personality & learning style)
- 🗺️ Personalized 8–10 year career roadmap
- 💬 24/7 AI counsellor chatbot
- 📊 Skill intelligence dashboards for schools & corporates

---

## 🧩 Modules

| Module | Description |
|------|------------|
| Career Recommendation Engine | Predicts best career options using ML |
| DMIT Dashboard | Analyzes brain dominance & learning style |
| Career Roadmap Generator | Step-by-step future career plan |
| AI Counsellor Chatbot | Instant career guidance anytime |
| Skill Intelligence Analytics | Performance & skill-gap analysis |

---

## 🤖 Machine Learning Model

- Algorithm: **Random Forest**
- Training Data: **500+ student profiles**
- Accuracy: **80%+**
- Input Parameters: Aptitude, interests, skills, personality traits
